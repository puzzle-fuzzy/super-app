# API 参考

Super API 是 RESTful 风格，基础路径为 `/api`。

## 通用规范

**认证**：大部分接口需要登录。使用 HttpOnly Cookie (`super.sid`) 或 Bearer Token (API Key)。

**响应格式**：

```json
// 成功
{ "success": true, "data": { ... } }

// 列表
{ "success": true, "items": [...], "total": 42 }

// 失败
{ "success": false, "error": "ERROR_CODE", "message": "..." }
```

**错误码**：

| 码 | 含义 |
|----|------|
| `UNAUTHORIZED` | 未登录 |
| `FORBIDDEN` | 无权限 |
| `VALIDATION_ERROR` | 参数校验失败 |
| `NOT_FOUND` | 资源不存在 |
| `CONFLICT` | 资源冲突 |
| `RATE_LIMITED` | 请求过于频繁 |
| `INTERNAL_ERROR` | 服务器内部错误 |
