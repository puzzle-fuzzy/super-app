# Product

## Register

product

## Users

Super is for individual creators and small creative teams who sign in once, then move between focused apps for assets, canvas work, transfers, API usage, and future generation workflows. The assets app is a personal creative library, used while collecting reusable files, prompts, subjects, and production material.

## Product Purpose

Super provides a unified cloud workspace where apps share one account, one API, one asset system, and one deployment model. The assets surface should help users quickly collect, recognize, preview, edit, and reuse creative material without feeling like they are operating an internal admin console.

## Brand Personality

Calm, capable, creative. The interface should feel polished and user-facing, with enough warmth and atmosphere to support creative work while staying clear and efficient for daily use.

## Anti-references

Do not make authenticated apps feel like database dashboards, CRUD admin panels, or generic SaaS control rooms. Avoid table-first layouts, loud enterprise metrics, decorative gradients without purpose, nested cards, and copy that speaks to operators instead of creators.

## Design Principles

1. Lead with the user's creative library, not the system model.
2. Make common actions visible: upload, create text, create subject, preview, edit, delete.
3. Let assets feel like reusable creative objects, not rows in storage.
4. Keep structure familiar and trustworthy, with restrained motion and clear states.
5. Preserve shared platform consistency while allowing each app to express its task.

## Accessibility & Inclusion

Target WCAG AA contrast for text and controls. Support keyboard focus, readable Chinese labels, reduced motion preferences, and responsive layouts that keep primary actions reachable on smaller screens.
